# 🚀 Estudos de JavaScript

Repositório destinado ao armazenamento de exercícios, desafios e projetos práticos realizados durante meus estudos de JavaScript, com foco em evolução para o ecossistema Node.js.

## 📌 Sobre o Projeto
Este repositório serve como um log de aprendizado. Aqui organizo o progresso das aulas, desde a lógica de programação básica até conceitos avançados de Back-end.

## 📂 Estrutura de Pastas

Atualmente, o repositório conta com **25 aulas** (e contando!):
* `aula-01-a-25/`: Fundamentos, sintaxe básica e manipulação de dados.

## 🛠️ Tecnologias Utilizadas
* **JavaScript (ES6+)**
* **Node.js**
* **Git & GitHub** (Controle de versão)
